# Consolidated preseed root

This directory is the single physical root for orchestration/compiler preseeds used by `ikam-perf-report` development and tests.

Layout:

- `declarations/` - executor and operator declarations
- `workflows/` - authored workflow sources
- `operators/` - operator inputs and related definitions
- `compiled/` - compiled graph outputs generated by the compiler pipeline
- `support/` - auxiliary support files such as JSON registries or lookup data

Rules:

- physical files live under this one root
- logical preload pipelines remain separate
- compiled outputs are derived artifacts, not the source of truth
- the compiler pipeline replaces static hand-maintained compiled YAML over time
